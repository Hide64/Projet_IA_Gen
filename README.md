# Projet_IA_Gen
Concevoir et développer une application innovante exploitant les technologies et modèles d’IA générative. L’objectif est d’apprendre à utiliser et manipuler des modèles tels que les LLM (Large Language Models) ou d’autres modèles génératifs (image, vidéo, voix, musique, etc.) tout en appliquant de bonnes pratiques de développement logiciel.
